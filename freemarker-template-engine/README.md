Wrapper of Java Freemarker 2.3 template engine as
[JSR 223](https://www.jcp.org/en/jsr/detail?id=223) compatible
[script engine](http://docs.oracle.com/javase/7/docs/api/javax/script/ScriptEngine.html).

You can use the template engine in CadenzaFlow Automation Platform as scripting language.

## More template engines (Community Extensions)

You can find the following template engine extensions in CadenzaFlow's [Community Hub](https://github.com/cadenzaflow-community-hub/cadenzaflow-7-template-engines-jsr223/):

* Velocity 2.3
* xQuery Saxon 9.6
* XSLT Saxon 9.6
